## Workflow

The application operates on a simple three-step pipeline:

1. **Target**: Define the directory path for modification.
2. **Source**: Provide the high-resolution image file.
3. **Deploy**: Execute the native icon transformation.

## Technical Specifications

- **Engine**: Swift 5.9
- **Frameworks**: SwiftUI, AppKit
- **Deployment Target**: macOS 14.0+
- **Security**: Sandboxed for secure file system interaction
